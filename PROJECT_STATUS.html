What you can do right now

Open a Claude Code session in this folder and say anything like:

  • "Write a strategy that buys NO on any market where implied probability is above 80%"
  • "Test momentum but with a 3-hour window instead of 6"
  • "What if we fade any market that gaps up more than 15% in a single tick?"
  • "Compare Kelly at quarter-fraction vs half-fraction on the same markets"

Claude will read CLAUDE.md, write the strategy file, run a backtest, show the scored result, and ask what you want to try next.

Or use the Python API directly:

from predmarkets import run_backtest, show_result
from predmarkets.strategies.kelly_sizing import KellySizing

strategy = KellySizing(
    p_oracle=lambda m: 0.7 if m.yes_price < 0.5 else 0.3,
    assigned_capital=10_000,
)
result = run_backtest(strategy, markets=10, bankroll=10_000, save_as="first_try")
show_result(result, bankroll=10_000)
